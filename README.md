# Blogs Application

## Please follow the steps below for setting up project.

Clone the Repository.

`git clone https://github.com/kstomar/blog-post`


Install all the dependencies.

`ruby - 3.0.0 `

`rails - 7.1.3`

Install gems.

`bundle install`

Database setup.

`rails db:create `

`rails db:migrate`

Run server.

`rails s`

Vist the url on a browser to run the application.
http://localhost:3000
